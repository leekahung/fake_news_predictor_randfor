Fake News Predictor

This web application for fake news prediction are currently coded only for .csv files at this time with a max limit of 25 MB of data (see main GitHub repository). The csv file must contain the follow columns or it won't work: 'id', 'title', 'author', and 'text' (see formatted examples from repository under the names test.csv and train.csv). The column 'label' is optional, but would be good to have if you wish to compare actual with predicted results. The model being used for this web application is Random Forest with an accuracy of 82%.
